# AI Chat 앱

## 구조

- [server](apps/server)
- [web](apps/web)

## 실행하기

### 사전 준비

- Node.js
- npm
- Docker

### server 실행

```bash
npm i
npm run start:dev --workspace=@atjsh/ai-chat-server
```

### web 실행

```bash
npm i
npm run dev --workspace=@atjsh/ai-chat-web
```
